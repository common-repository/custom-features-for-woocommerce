A plugin of some custom features for your WooCommerce site.

== Description ==

Sometimes WooCommerce default features may be not fulfill your requirements, So you can use <strong>Custom Features for WooCommerce</strong> plugin to 
get some extra features like as bellow.

* When your store is not available for delivery, you can set your store as close mode. 
* Cart data will be automatically updated when product quantity is change.

== Installation ==

After login as administrator, go to <strong>Plugin > Add New</strong>.

== Frequently Asked Questions ==

= How to setup store as close? =

After install and activate <strong>Custom Features for WooCommerce</strong> go to <strong>Custom Features for WooCommerce</strong> > <strong>Store close</strong> > Setup all field.

= How to setup cart data automatically updated when product quantity is change? =

After install and activate <strong>Custom Features for WooCommerce</strong> go to <strong>Custom Features for WooCommerce</strong> > <strong>Cart auto update</strong> > choose <strong>Yes</strong>.
